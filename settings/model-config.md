# Agent团队模型配置

本文档记录团队各成员角色所使用的大模型配置。

---

## 配置说明

每个角色可以配置不同的大模型，以发挥各自的优势：
- **总编**：需要理解能力强，适合复杂推理
- **调研员**：需要知识面广，搜索能力强的模型
- **写手**：需要创意丰富、风格多样的模型

## 使用方法

### 首次配置

请使用 `/config-model` 命令配置各角色的大模型。

### 查看当前配置

请使用 `/show-models` 命令查看当前配置。

### 调整配置

请使用 `/config-model [角色] [模型名]` 命令调整特定角色的模型。
