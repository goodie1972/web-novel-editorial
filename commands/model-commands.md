# 模型配置命令

本文档定义团队成员的大模型配置命令。

---

## 命令列表

### --config-model

配置团队成员使用的大模型。

**用法**：
```
--config-model [角色] [模型名]
```

**参数**：
- `[角色]`：要配置的角色（总编/调研员/写手）
- `[模型名]`：要使用的大模型名称

**示例**：
```
--config-model 总编 claude-opus-4-20250514
--config-model 写手 gpt-4-turbo
--config-model 调研员 gemini-1.5-pro
```

---

### --show-models

查看当前各角色的大模型配置。

**用法**：
```
--show-models
```

**示例输出**：
```
当前模型配置：
- 总编：claude-opus-4-20250514
- 调研员：gemini-1.5-pro  
- 写手：gpt-4-turbo
```

---

### --list-models

列出所有可用的大模型。

**用法**：
```
--list-models
```

---

## 首次配置提示

**当系统检测到模型配置未完成时，应提示用户进行配置：**

```
⚠️ 模型配置未完成！

在使用团队之前，请先配置各成员使用的大模型。

请使用以下命令配置：

--config-model 总编 [模型名]
--config-model 调研员 [模型名]  
--config-model 写手 [模型名]

或者一次性配置所有角色：
--config-model all [模型名]
```
