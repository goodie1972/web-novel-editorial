# 副本设计

本文档定义故事中的各类副本设计。

---

## 副本类型

| 类型 | 目的 | 爽点来源 | 典型场景 |
|------|------|----------|----------|
| **升级副本** | 主角实力提升 | 突破、收获 | 秘境探险、遗迹传承 |
| **危机副本** | 生死考验 | 逆袭、守护 | 宗门大比、魔渊入侵 |
| **收获副本** | 获得珍贵资源 | 收获、惊喜 | 宝藏、神兽、传承 |
| **情感副本** | 人物关系发展 | 甜蜜、感动 | 历练、红颜、结义 |
| **解密副本** | 揭开世界秘密 | 揭秘、反转 | 上古遗迹、历史真相 |
| **复仇副本** | 解决恩怨 | 打脸、碾压 | 报仇雪恨 |

## 副本节奏

### 单副本节奏

```
进入副本 → 遭遇小怪 → 发现机遇 → 遇到强敌 → 激战 → 获胜/突破 → 收获 → 离开
```

### 全书副本节奏

```
副本1（简单）→ 副本2（中等）→ 副本3（困难）→ 副本4（简单）→ ... → 终极副本（生死级）
```
