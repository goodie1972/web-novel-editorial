# 战力数值设定

本文档定义详细的战力数值体系，用于战斗场景的量化计算。

**注意**：仅战斗类作品需要此文件。

---

## 战力计算公式

### 基础战力

```
综合战力 = 基础战力 × 技能加成 × 装备加成 × 血脉加成 × 环境因素
```

### 各项系数

| 系数 | 范围 | 说明 |
|------|------|------|
| 基础战力 | 1-10000 | 等级/境界决定 |
| 技能加成 | 1.0-10.0 | 技能威力 |
| 装备加成 | 1.0-5.0 | 武器防具 |
| 血脉加成 | 1.0-3.0 | 血脉觉醒度 |
| 环境因素 | 0.5-2.0 | 主场/客场 |

## 战斗数值规则

### 胜负判定

| 战力差距 | 结果 |
|----------|------|
| <20% | 胜负难料 |
| 20%-50% | 弱势方可坚持 |
| 50%-100% | 碾压 |
| >100% | 一击必杀 |

### 跨越战斗

| 跨越幅度 | 合理性 |
|----------|--------|
| 1级 | 主角可越级 |
| 2级 | 需要金手指/底牌 |
| 3级+ | 需要主角光环/剧情杀 |
