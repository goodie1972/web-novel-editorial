> 
> 最后一个字落下时，人已消失在原地。

### 示例：导师/引路人

| 字段 | 内容 |
|------|------|
| 姓名 | 赵云山 |
| 性别/年龄 | 男，60岁 |
| 身份 | 超管局局长 |
| 外貌 | 身材高大，国字脸，浓眉下一双虎目不怒自威，常穿黑色中山装 |
| 性格 | 外冷内热，护短，重情义 |
| 说话风格 | 洪亮有力，喜欢用命令语气，但关心人时会放轻声音 |
| 与主角关系 | 亦师亦父 |
| 核心欲望 | 守护人类安宁 |
| 恐惧 | 失去重要后辈 |
| 在剧情中的作用 | 引路人，提供情报，在主角危难时出手 |
| 结局 | 为救主角重伤陨落 |

### 示例对话

**场景：指导主角修炼**

> "集中注意力！感受灵气在经脉中流动，像水流一样，不要急。"
> 
> 赵云山站在身后，大手按在林渊背上，浑厚的真气渡入体内。
> 
> "就是这里，冲过去，不要停！"
> 
> 声音如洪钟，却在关键时候压低了几分。
> 
> "成了。"

## 角色一致性检查清单

每次写作前必须检查：

- [ ] 主角外貌是否一致（身高/眼神/标志性特征）
- [ ] 主角说话风格是否一致（短句/语气词）
- [ ] 配角性格是否一致
- [ ] 实力等级是否合理
- [ ] 人物关系是否有变化
- [ ] 称呼是否正确
- [ ] 口头禅是否正确
