## 描写规范

### 正面清单（应该这样做⭐）

| 类型 | 要求 | 示例 |
|------|------|------|
| 感官描写 | 多用视/触/嗅/听 | 详见下方 |
| 对话 | 简练，有潜台词 | 详见下方 |
| 心理 | 细腻但不过度 | 详见下方 |
| 动作 | 具体，有画面感 | 详见下方 |

### 负面清单（禁止这样做⭐）

| 类型 | 禁止 | 替代 |
|------|------|------|
| 形容词堆砌 | "他是一个非常英俊潇洒..." | 用动作描写 |
| 直接陈述 | "他很生气" | 用细节表现 |
| 说教对话 | "我要告诉你..." | 让角色自己说 |
| 心理描写过长 | 大段内心独白 | 用表情/动作代替 |

## 感官描写指南

### 视觉

**应该**：
- 颜色、光影、形状的具体描写
- 动静对比
- 细节聚焦

**示例**：
> 夕阳把云层染成血红，一道金光穿透云隙，落在山谷中的宗门大殿上琉璃瓦，反射出刺眼的光芒。

**禁止**：
- 笼统的"很美"、"很漂亮"

### 听觉

**应该**：
- 具体的声音
- 声音的变化
- 声音与情绪的对应

**示例**：
> 钟声浑厚，在山谷间回荡三声，余音未散，又是一阵急促的脚步声。

### 触觉/其他

**应该**：
> 灵气入体，像温泉流过经脉，暖暖的，又有点麻痒。

## 对话规范

### 对话原则

| 原则 | 说明 |
|------|------|
| 潜台词 | 角色说话要有言外之意 |
| 性格化 | 不同角色说话方式不同 |
| 信息节制 | 不要一次性把所有信息都说出来 |

### 正面示例

**老王头（老江湖，，说话带方言）**：
> "小子，你当灵气是大风刮来的？还嫌慢？"
> 
> 他瞥了林渊一眼，嘴角抽了抽，
> 
> "当年老子光是引气入体，就花了三个月。你这速度，已经够快了 知不知道？"
> 
> 顿了顿，又补了一句：
> 
> "别好高骛远，脚踏实地才是正经。"

**主角（沉默寡言，性格隐忍）**：
> "嗯。"
> 
> 他惜字如金，只从鼻子里挤出一个字。
> 
> 目光却始终盯着那枚玉简，手指微微蜷曲，显示内心并不如表面平静。

### 负面示例（禁止）

> "林渊，我告诉你，这件事情的真相是这样的..."
> "好的，我明白了，谢谢你的解释。"

## 心理描写规范

### 原则

| 原则 | 说明 |
|------|------|
| 用细节代替直接陈述 | "他很紧张" → 细节表现 |
| 情绪要有层次 | 愤怒可以夹杂着恐惧、悲伤 |
| 与情节关联 | 心理变化推动情节发展 |

### 正面示例

> 林渊瞳孔微缩。
> 
> 恐惧？
> 
> 不，是兴奋。
> 
> 那种血液沸腾的感觉又来了，像是一头被困已久的野兽，终于嗅到了猎物的气息。
> 
> 手指无意识地摩挲着剑柄，他在克制，克制着现在就冲出去的冲动。

### 负面示例（禁止）

> "林渊感到非常愤怒，他决定要报复对方。"（直接陈述）
> 
> "他心里有些紧张，有些害怕，还有些期待。"（情绪杂糅，没有重点）

## 动作描写规范

### 原则

| 原则 | 说明 |
|------|------|
| 具体化 | "他打架" → 具体动作 |
| 分解动作 | 一个动作拆成多个细节 |
| 与情绪关联 | 动作反映心理状态 |

### 正面示例

> 切磋开始！
> 
> 林渊左脚前踏，踩在青石板上，发出"咔嚓"一声脆响。
> 
> 腰部拧转，右拳带起风声，直取对方面门。
> 
> 这一拳用了七分力，带起的气流掀起地面的落叶，在阳光下形成一个黄色的漩涡。

## 禁用词汇表

### AI高频词（必须替换⭐）

| 禁用词 | 替换方案 |
|--------|----------|
| 仿佛 | 像、似乎、宛如 |
| 心中一动 | 瞳孔微缩、动作一顿 |
| 冷笑一声 | 嘴角翘起、发出不屑 |
| 不置可否 | 没有回答、沉默 |
| 综上所述 | 总之、总而言之 |

### 流水账句式

| 禁止 | 改进 |
|------|------|
| "然后...然后...然后..." | 用具体动作连接 |
| "他走到...他看到...他想到..." | 合并动作，突出重点 |
