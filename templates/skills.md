# 技能设定模板

## 超能力等级体系

```
F → E → D → C → B → A → S → SS → SSS → 神级 → 超神级 → ...
（每级分为：初期、中期、后期、巅峰）
```

## 主角技能

### 核心技能：时空之刃

| 属性 | 内容 |
|------|------|
| 名称 | 时空之刃 |
| 等级 | SSS级（可进化） |
| 类型 | 特殊类/进化类 |
| 能力 | 操控时空 |
| 进化路线 | 见下方 |

## 修炼体系

### 修仙境界

```
炼体 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 仙人
（每境界9层）
```

### 双体系关系

| 体系 | 等级对应 | 特点 |
|------|----------|------|
| 超能力 | F→SSS | 觉醒获得 |
| 修仙 | 炼体→仙人 | 需要修炼 |
| 融合 | 两者结合 | 更强 |

## 技能平衡检查

每次战斗后检查：
- [ ] 主角实力是否合理
- [ ] 敌人实力是否合理
- [ ] 越级战斗是否有足够理由

- [ ] 技能威力是否一致
