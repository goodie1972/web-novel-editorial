# 章节规划模板

## 使用说明

本模板用于规划每5章的内容。每个阶段（5章）必须有明确的：
- 爽点设计
- 章节目标
- 字数要求
- 伏笔埋设
- 钩子设计

**重要**：每个5章批次必须有详细的章纲（一句话大纲），不能只有章节名！

## 详细章纲模板

**核心理念**：拒绝"盲写"，每个章节必须有详细的场景列表。

### 第X章：章节名

**章纲（一句话）**：xxx

**剧情走向**：
- 开篇（承接上章钩子）：xxx
- 发展：xxx
- 高潮：xxx
- 结尾（钩子）：xxx

### 场景拆分（必须⭐）

**这是最关键的一步！不要让AI"自由发挥"，必须拆分场景**

| 场景 | 地点 | 时间 | 人物 | 事件 | 重点 |
|------|------|------|------|------|------|
| 场景1 | | | | | |
| 场景2 | | | | | |
| 场景3 | | | | | |
| 场景4 | | | | | |

**示例（正确格式）**：

### 第5章：灵泉异变

**章纲（一句话）**：主角在后山修炼时发现灵泉异变，触发神秘传承

**场景拆分**：

| 场景 | 地点 | 时间 | 人物 | 事件 | 重点 |
|------|------|------|------|------|------|
| 场景1 | 杂役院 | 清晨 | 主角 | 主角醒来，通过环境描写体现压抑感 | 营造压抑氛围 |

| 场景5 | 后山 | 夜晚 | 主角 | 主角回到住处，思考对策，甩出下一章钩子 | 章尾钩子 |

### 场景详细说明

每个场景需要填写：

```
### 场景1：清晨起床
- **地点**：杂役院弟子房
- **时间**：清晨，天刚亮
- **人物**：林渊（主角）
- **事件**：
  1. 主角从硬板床上醒来
  2. 通过环境描写体现杂役院的简陋：霉味的被子、漏风的窗户
  3. 主角内心思考今天的计划
- **重点**：
  - 用环境描写体现身份低微
  - 主角的性格：沉稳、不抱怨
- **字数**：约300字
```

## 示例：第21-25章

### 第21-25章：超管局崛起

| 章节 | 章纲（一句话） | 爽点 | 钩子 | 字数 | 状态 |
|------|----------------|------|------|------|------|
| 21 | 主角接受超管局邀请，决定加入 | 加入编制 | 超管局内部有卧底？ | 2500 | 待写 |
| 22 | 主角首战，秒杀S级怪物 | 越级秒杀 | 怪物体内有神秘芯片？ | 2500 | 待写 |
| 23 | 主角卧底黑暗议会被发现 | 身份暴露危机 | 原来叛徒是... | 2500 | 待写 |
| 24 | 主角破坏深渊之门，拯救世界 | 拯救世界 | 通道背后是... | 2500 | 待写 |
| 25 | 主角突破SS级，震动全球 | 突破+震惊 | 异变突生！ | 2500 | 待写 |

**本阶段目标**：主角加入超管局，获得官方身份，开始崭露头角

**本阶段爽点**：
1. 第21章：加入超管局，获得编制（身份爽）
2. 第22章：首战秒杀S级怪物（碾压爽）
3. 第24章：破坏深渊之门拯救世界（保护爽）

**本阶段伏笔**：
- 神秘芯片（第22章埋下，第35章回收）
- 卧底身份（第23章埋下，第40章回收）
- 通道背后的秘密（第24章埋下，第80章回收）

**本阶段钩子**：
- 章节钩子：每章结尾都有悬念
- 批次钩子：25章结尾"异变突生"引出后续剧情

**人物出场**：
- 赵云山（第21章）- 超管局局长
- 新反派A（第23章）- 黑暗议会卧底

**风险提示**：
- 黑暗议会实力强大，可能有更深的阴谋
- SS级突破可能引发天地异象

## 质量检查清单

每个阶段完成后，检查：

- [ ] 5章全部完成
- [ ] 每章字数2500-3000
- [ ] 每章至少1个爽点
- [ ] 每章结尾有钩子
- [ ] 有详细的章纲（一句话大纲）
- [ ] 伏笔已埋设
- [ ] 与大纲一致
- [ ] 人物性格一致
- [ ] 战力体系一致
