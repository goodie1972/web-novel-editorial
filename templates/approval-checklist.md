# 大纲定稿确认清单

## 定稿前必须完成

在开始写作之前，必须完成以下所有文档：

### 1. 大纲 (outline.md)

- [ ] 主线剧情清晰
- [ ] 每10章有明确目标
- [ ] 爽点布局合理（每10章至少3个）
- [ ] 伏笔埋设计划
- [ ] 结局设计

### 2. 世界观 (world.md)

- [ ] 时代背景
- [ ] 地理设定
- [ ] 等级体系（超能力+修仙）
- [ ] 势力分布
- [ ] 规则设定
- [ ] 自洽性检查

### 3. 人物设定 (characters.md)

- [ ] 主角完整设定（姓名、性格、成长线）
- [ ] 主要配角完整设定
- [ ] 反派完整设定
- [ ] 人物关系图
- [ ] 角色一致性检查

### 4. 技能设定 (skills.md)

- [ ] 主角核心技能
- [ ] 技能进化路线
- [ ] 等级体系对应
- [ ] 敌人技能
- [ ] 技能平衡检查

### 5. 章节规划 (chapter-plan.md)

- [ ] 第1-5章详细规划
- [ ] 第6-10章详细规划
- [ ] 第11-15章详细规划
- [ ] ...（每5章一组）
- [ ] 每章爽点明确
- [ ] 每章字数要求（2500-3000）
- [ ] 伏笔埋设计划
