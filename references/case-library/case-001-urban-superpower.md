# 案例一：都市异能《超能进化》

## 项目概述

| 项目 | 内容 |
|------|------|
| 书名 | 《超能进化》 |
| 题材 | 都市异能 |
| 风格 | 系统流 + 升级 |
| 目标平台 | 起点中文网 |
| 预计字数 | 100万字 |
| 实际字数 | 25万字（进行中） |

---

## 用户需求

> 想写一本都市异能小说，主角有超能力系统，从普通人变成强者。风格要爽快，不要太虐。目标是起点签约。

---

## 研究简报摘要

### 榜单调研结论

起点都市异能榜单特点：
- 系统流占比 60%+
- 主角性格偏好：杀伐果断
- 金手指类型：进化系统最热门
- 节奏：前10章必须有3次爽点

### 拆文分析

**拆解作品**：《全球进化》

| 维度 | 分析 |
|------|------|
| 开篇 | 第1章立人设+系统觉醒，第2章首战，第3章突破 |
| 爽点 | 每2章一个爽点，叠加旁观者震惊 |
| 金手指 | 进化系统，可吞噬怪物升级 |
| 成功要素 | 节奏快、爽点密、主角不憋屈 |

### 决策1：金手指类型

| 选项 | 最终选择 | 理由 |

## 成功经验

1. **黄金三章**：第1章系统觉醒+冲突引入，第2章首战，第3章突破+钩子
2. **爽点密度**：平均每2章一个爽点，读者反馈"停不下来"
3. **主角性格**：不憋屈、不圣母，符合起点读者口味

---

## 遇到的问题

1. **中期节奏放缓**：第15-18章爽点减少，读者反馈"有点水"
   - 解决：增加突发事件，压缩过渡章节
2. **配角存在感弱**：配角没有记忆点
   - 解决：给配角增加口头禅和独特外貌

---

## 经验沉淀

| 类型 | 经验 |
|------|------|
| 写作 | 每5章检查爽点数量，不足则补充 |
| 审核 | 编辑重点检查"憋屈感"，超过3章必须打回 |
| 设定 | 等级体系不要太复杂，读者记不住 |
