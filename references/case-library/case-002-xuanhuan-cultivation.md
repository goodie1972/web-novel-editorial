# 案例二：玄幻修仙《万古剑尊》

## 项目概述

| 项目 | 内容 |
|------|------|
| 书名 | 《万古剑尊》 |
| 题材 | 玄幻修仙 |
| 风格 | 传统修仙 + 穿越 |
| 目标平台 | 番茄小说 |
| 预计字数 | 200万字 |
| 实际字数 | 50万字（进行中） |

---

## 用户需求

> 想写一本传统玄幻小说，主角穿越到修仙世界，从废柴逆袭成为剑道至尊。要有多重身份反转，打脸爽。目标是番茄签约。

---

## 研究简报摘要

### 榜单调研结论

番茄玄幻榜单特点：
- 穿越+废柴逆袭占比最高
- 主角需要"双重身份"制造反转
- 节奏要求快，前5章必须有打脸
- 女主线可以多条，但不要拖沓

### 拆文分析

**拆解作品**：《剑道第一仙》

| 维度 | 分析 |
|------|------|
| 开篇 | 穿越+废柴开局，第3章隐藏身份首次展露 |
| 爽点 | 打脸为主，叠加"众人震惊"效果 |
| 设定 | 修炼体系清晰，每境九重 |
| 成功要素 | 身份反转刺激、主角深藏不露 |

---

## 核心设定

### 主角

| 字段 | 内容 |
|------|------|
| 姓名 | 林轩 |
| 原身份 | 地球剑道宗师 |
| 现身份 | 林家废柴少爷（表面） |
| 真实身份 | 剑帝转世 |
| 性格 | 深藏不露，一剑封喉 |

### 修炼体系

```
炼体 → 筑基 → 金丹 → 元婴 → 化神 → 渡劫 → 真仙
每境：初期、中期、后期、巅峰
```

### 势力设定

| 势力 | 立场 | 与主角关系 |
|------|------|------------|
| 林家 | 表面阵营 | 主角出生地，被排挤 |
| 剑宗 | 核心阵营 | 主角前世创建 |
| 天魔宗 | 敌对 | 最终Boss所在 |
| 隐世家族 | 中立 | 传承来源 |

---

## 关键决策记录

### 决策1：开篇设计

| 选项 | 最终选择 | 理由 |
|------|----------|------|
| 直接无敌 | ✗ | 缺乏成长感 |
| 废柴逆袭 | ✓ 选择 | 经典模式，番茄读者接受度高 |
| 天才坠落 | ✗ | 节奏太慢 |

### 决策2：感情线

| 选项 | 最终选择 | 理由 |
|------|----------|------|
| 单女主 | ✗ | 用户希望多条感情线 |
| 双女主 | ✓ 选择 | 一个青梅竹马+一个天才少女 |
| 后宫 | ✗ | 番茄审核风险 |

---

## 爽点设计

### 反转爽设计

```
第5章：众人嘲讽主角"废物"
第6章：主角一剑秒杀强敌
第7章：众人震惊"他是谁？"
第8章：主角恢复"废物"状态，众人怀疑人生
```

### 打脸节奏

| 章节 | 打脸对象 | 方式 |
|------|----------|------|
| 5章 | 族内天才 | 比武碾压 |
| 15章 | 宗门长老 | 隐藏实力暴露 |
| 30章 | 敌对宗门 | 一剑灭宗 |

---

## 成功经验

1. **身份反转**：主角每次展露实力后迅速隐藏，制造持续的"打脸期待"
2. **伏笔布局**：前期埋下"剑帝传承"伏笔，中期揭秘
3. **配角记忆点**：给配角独特的口头禅和外貌特征

---

## 遇到的问题

1. **等级体系混乱**：前期设定不够详细，后期出现战力崩坏
   - 解决：创建 combat.md 详细记录战力数值
2. **伏笔回收不及时**：埋下的伏笔忘记回收
   - 解决：在 foreshadowing.md 中跟踪每个伏笔状态

---

## 经验沉淀

| 类型 | 经验 |
|------|------|
| 设定 | 等级体系要在开篇就定死，不能后期修改 |
| 伏笔 | 每埋一个伏笔就记录回收章节，定期检查 |
| 节奏 | 每20章检查整体方向，防止"写飞" |
