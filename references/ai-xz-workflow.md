# 爱写作(AI-XZ) 创作流程详细调研

> 网站：https://www.ai-xz.com
> 调研日期：2025-02-21

---

## 整体架构：四步创作法

```
故事信息设定 → 作品设定完善 → 章节大纲生成 → 章节内容创作
```

---

## 第一步：故事信息设定

### 1.1 标题与类型选择

**功能**：
- 完整的小说类型选择器
- 主类型：玄幻、都市、历史、科幻、悬疑、二次元等
- 子类型：东方玄幻、西方魔幻、异界大陆等

**意义**：准确的类型定位影响后续AI生成内容的风格、用词和情节走向

### 1.2 故事梗概（核心驱动力）

**要求**：不是简单概括，需详细描述
- 主要角色
- 背景设定
- 核心冲突
- 故事走向
- 关键情节点

**特色功能**：
- "AI灵感"按钮：根据已输入信息提供创意建议
- 人机协作方式，保持创作个人特色

**重要性**：梗概将成为后续所有AI生成内容的重要参考

### 1.3 世界观与力量体系

**世界观设定**：
- 历史背景
- 地理环境
- 政治制度
- 社会结构

**力量体系**：
- 境界划分
- 修炼方法
- 职业分类

**特色**：每个部分配有"AI生成"按钮，可生成建议内容并自由编辑

---

## 第二步：作品设定完善

### 2.1 势力组织构建（非必填但重要）

**内容**：
- 宗门/公司/星际联盟等组织架构
- 组织间关系网络
- 地域分布
- 实力对比

**AI辅助**：根据世界观设定智能生成符合逻辑的势力组织

### 2.2 金手指设定

**维度**：
| 维度 | 说明 |
|------|------|
| 获得方式 | 如何获得金手指 |
| 能力表现 | 金手指的具体能力 |
| 成长机制 | 如何升级/进化 |
| 使用限制 | 限制条件（避免破坏平衡）|

**AI辅助**：考虑金手指与整体故事设定的协调性，避免逻辑漏洞

### 2.3 角色设定智能化管理

**核心功能**：
- "从梗概提取主角"：自动分析梗概识别主要角色
- 建立角色基础档案
- 初步分析性格特点、关系网络、故事作用

**智能化程度**：
- 识别角色姓名
- 分析性格特点
- 建立关系网络

**记忆功能**：后续章节生成保持角色设定记忆，确保一致性

---

## 第三步：章节大纲生成

### 3.1 提示词模板

**预设模板**：
- 多种大纲生成提示词模板
- 每个模板针对不同创作需求和风格优化

**自定义功能**：
- 可选择预设模板
- 可自定义修改提示词内容

**参数化输入**：
- 指定章节数量
- 指定每章字数
- 指定情节密度

### 3.2 多模型选择

**支持的模型**：
| 模型 | 特点 |
|------|------|
| Claude | 擅长情节构思 |
| GPT | 综合能力强 |
| Gemini | 逻辑严谨 |
| DeepSeek | 推理能力 |
| Kimi | 长文本处理 |
| Qwen | 中文优化 |
| Grok | 创意丰富 |

**选择策略**：根据创作风格和当前需求选择最适合的模型

### 3.3 大纲编辑

**功能**：
- 实时编辑
- 调整章节顺序
- 修改情节安排
- 增删内容

**智能提示**：选中困难文本提供针对性创作建议

**格式化功能**：自动整理大纲结构，使其清晰易读

---

## 第四步：章节内容创作

### 4.1 无缝衔接

**自动调用**：之前设定的所有内容成为创作参考
- 人物关系
- 世界观
- 力量体系
- 情节大纲

### 4.2 AI记忆库（核心特色）

**功能**：
- 记住之前章节的情节细节
- 记住人物行为
- 记住对话风格
- 记住法宝/物品设定

**一致性保障**：确保整部小说的连贯性

### 4.3 实时协助功能

| 功能 | 说明 |
|------|------|
| AI追问 | 针对当前情节提出问题，理清思路 |
| 智能润色 | 选中文本提供改写建议 |
| 情节建议 | 提供情节发展建议 |
| 对话灵感 | 提供角色对话灵感 |
| 场景描写 | 描述场景，AI生成丰富描写 |

---

## 辅助功能

### AI群聊头脑风暴

- 邀请多个AI模型进入聊天室
- 模型互相提问、辩论、补充
- 从多维度碰撞激发灵感

### AI全局助手

- 随时唤起
- 解答创作问题
- 提供专业建议

### AI拆书（需登录）

- 分析已有小说结构
- 提取设定要素

## 文档要求总结

### 必填项

| 文档 | 内容 |
|------|------|
| 标题 | 小说名称 |
| 类型 | 主类型+子类型 |
| 故事梗概 | 角色+背景+冲突+走向+情节点 |
| 世界观 | 历史+地理+政治+社会 |
| 力量体系 | 境界+修炼+职业 |

### 建议填写

| 文档 | 内容 |
|------|------|
| 势力组织 | 架构+关系+分布+实力 |
| 金手指 | 获得+能力+成长+限制 |
| 角色设定 | 从梗概提取+人工完善 |
| 章节大纲 | 提示词模板生成+编辑 |

### 创作过程

| 阶段 | 产出 |
|------|------|
| 大纲生成 | 详细章节规划 |
| 内容创作 | 章节正文+实时协助 |
| 一致性维护 | AI记忆库自动管理 |
