# 快速启动指南

5分钟了解如何使用网文编辑部。

---

## 第一步：提供需求

告诉我你想写什么：

```
题材：都市/玄幻/仙侠/历史/悬疑
风格：系统流/重生/穿越/传统
目标平台：起点/番茄/七猫/飞卢
预计字数：XX万字
```

**示例**：
> 我想写一本都市异能小说，系统流，主角有超能力系统，目标是起点签约，预计100万字。

---

## 第二步：等待研究

研究员会自动：
1. 调研目标平台榜单
2. 拆解热门作品
3. 产出创意建议

---

## 第三步：确认方案

全员讨论后，总编呈现方案：
- 书名建议
- 核心设定
- 章节规划

**你需要确认后才开始写作**

---

## 第四步：审核循环

每5章一批：
```
写手撰写 → 编辑审核 → 总编确认 → 通过/打回
```

每20章：
```
读者反馈 → 调整优化 → 继续
```

---

## 常用命令

| 需求 | 示例 |
|------|------|
| 创建项目 | "帮我创建一个玄幻小说项目，系统流，目标起点" |
| 查看进度 | "当前项目进度如何？" |
| 继续项目 | "继续上一个项目" |
| 调整设定 | "把主角改成女角色" |
| 修改审核标准 | "要求每章至少2个爽点" |

---

## 文件位置

项目文件在 `outputs/` 目录：
- `outline.md` - 大纲
- `characters.md` - 人物
- `chapters/` - 章节
- `reviews/` - 审核

---

## 最小示例

**输入**：
> 写一本都市系统签到流，主角每天签到变强，目标番茄签约，50万字

**输出**：
1. 研究简报（番茄签到流特点、热门作品拆解）
2. 创意建议（书名选项、金手指设计）
3. 大纲定稿（等你确认）
4. 章节撰写（逐批完成）

---

## 注意事项

1. **确认后再写**：大纲必须你确认后才开始写作
2. **审核暂停**：编辑审核期间不会继续写下一批
3. **反馈调整**：每20章会根据读者反馈调整
4. **人工审阅**：产出是高质量初稿，终稿仍需你审阅
