## 写作技巧

### 主角塑造（秦羽）

1. **重情重义**：为了父亲、兄弟可以拼命
2. **坚持不懈**：资质不好但勤奋努力
3. **正面形象**：不做作，不阴郁
4. **有责任感**：对家族、对亲人

### 主角塑造（林雷）

1. **执着**：认定目标就坚持到底
2. **重情**：对妻子、对儿子、对兄弟
3. **战斗天才**：战斗天赋极高
4. **正邪分明**：有原则，有底线

### 升级体系

#### 《星辰变》
```
后天（1-12层）→ 先天 → 渡劫 → 天仙 → 金仙 → 玄仙 → 仙帝 → 宇宙行者
```

#### 《盘龙》
```
战士：初级→中级→高级→圣域
魔法师：初阶→中阶→高阶→圣域
分成地、风、水、火、雷电、光明、黑暗七系
```

### 爽点设计

| 类型 | 出现场景 |
|------|----------|
| 升级突破 | 突破境界 |
| 越级杀敌 | 弱势击败强敌 |
| 获得传承 | 得到强者遗产 |
| 守护家人 | 为了亲人战斗 |

### 经典套路

1. **废物逆袭**：资质不好但靠努力成功
2. **奇遇连连**：不断获得宝物和传承
3. **越级挑战**：弱势挑战强者
4. **守护之战**：为了重要的人战斗
