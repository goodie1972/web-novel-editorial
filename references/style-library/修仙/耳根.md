### 场景：逆天改命（《仙逆》）

原文：
> “天要阻我，我便逆天！”
> 
> 王林抬起脚步，向着第一步的阶梯，轰然踏去！
> 
> 这一步落下，他的身体内传来轰鸣，这一步，似踏碎了某种枷锁，迈出了逆修的第一步！
> 
> 天地之间，有那么一道光，骤然降临！
> 
> 那光，落在王林身上，落在他的双眼，落在他的道袍，落在他的发丝！
> 
> 他整个人，在这一瞬间，仿佛与天地融合！
> 
> “天可逆，道可修，命……可改！！！”

原文：
> “兄弟，我走之后，你要好好活着！”
>

>

>

> 
> 王林抱着她，呆立原地。
> 
> 漫天风雪，似也在为这段感情哭泣。

## 写作技巧

### 主角塑造（《仙逆》王林）

1. **起点低**：从最底层崛起
2. **执念深**：为了目标可以付出一切
3. **重情义**：对师父、兄弟、爱人感情深厚
4. **杀伐果断**：该出手时绝不含糊

### 主角塑造（《一念永恒》白小纯）

1. **逗比**：性格跳脱，幽默风趣
2. **怕死**：虽然怕死但关键时刻很靠谱
3. **成长**：从搞笑到担当的成长线
4. **反转**：前期搞笑，后期热血

### 升级体系（《仙逆》）

```
凝气（1-13层）→ 筑基 → 结丹 → 元婴 → 化神 → 问鼎 → 阴虚 → 阳实 → 劫越 → 仙境
```

### 爽点设计

| 类型 | 出现场景 |
|------|----------|
| 逆袭 | 弱者逆天改命 |
| 守护 | 保护重要的人 |
| 执念 | 达成不可能的目标 |
| 悲壮 | 战斗到最后一刻 |

### 经典套路

1. **化凡**：主角体验凡人生活，感悟天道
2. **执念**：为一个人/一件事坚持到底
3. **逆天**：挑战天地规则
