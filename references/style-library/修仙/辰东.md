## 范本示例

### 场景：九龙拉棺（《遮天》）

原文：
> 九龙拉棺！
> 
> 那是一幅极其壮观的画面，在那无垠的星空下，九条庞然大物仿佛钢铁长城，横贯长空，绵延也不知道多少里。
> 
> 九条龙舟之后，是一个巨大的青铜古棺，当九龙与铜棺相连，划过星空时，发出的是金属的铿锵之音。
> 
> 青铜古棺之上，刻满了岁月的痕迹，有刀痕，有箭孔，镂刻着无尽古老的图案，似在诉说着那最为久远的传说。
> 
> 九龙拉棺，这是一幅震慑人心的画面，深深印在每一个看到这一幕的人心中。

## 写作技巧

### 主角塑造（《遮天》叶凡）

1. **豪情万丈**：天帝之气，镇压一切
2. **重情重义**：对兄弟、爱人、弟子深情
3. **杀伐果断**：对敌人绝不留情
4. **无敌信念**：同阶无敌，越阶可战

### 主角塑造（《完美世界》石昊）

1. **战天斗地**：一生都在战斗
2. **重情重义**：对亲人、故土的眷恋
3. **悲情英雄**：为了后世太平牺牲自己
4. **霸气侧漏**：还是熊孩子时就敢挑战一切

### 升级体系（《遮天》）

```
轮海（苦海→命泉→神桥→彼岸）→ 道宫（五脏）→ 四极（四肢）→ 化龙（脊椎）→ 仙台（1-9层）
→ 大圣 → 准帝 → 大帝 → 成仙
```

### 爽点设计

| 类型 | 出现场景 |
|------|----------|
| 碾压 | 同阶无敌，越级败敌 |
| 守护 | 护亲友、护种族 |
| 开拓 | 开创修炼体系 |
| 牺牲 | 悲壮之战 |

### 经典套路

1. **九龙拉棺**：宏大开头，悬念十足
2. **禁區探險**：生命禁区、帝陵、上古秘辛
3. **帝路争雄**：年轻天骄争霸
4. **守护之战**：守护族人、守护苍生
