## 文风特点

### 核心风格

| 特点 | 说明 |
|------|------|
| 稳健谨慎 | 主角韩立行事低调，不冒险，每步都有后手 |
| 逻辑严密 | 修真体系设定严谨，等级分明 |
| 冷静客观 | 主角很少有情绪波动，理性至上 |
| 资源为王 | 强调灵石、功法、资源的重要性 |

### 叙事特点

| 维度 | 特点 |
|------|------|
| 节奏 | 慢热型，铺垫长但扎实 |
| 爽点 | 越级挑战+资源收获 |
| 战斗 | 注重法宝、功法克制 |
| 心理 | 主角内心活动丰富但不冗余 |

### 语言风格

| 类型 | 示例 |
|------|------|
| 对话 | 简洁有力，少废话 |
| 描写 | 环境描写较少，重点在修炼和宝物 |
| 心理 | 主角思考以利益计算为主 |

## 范本示例

### 场景：突破境界

原文：
> 韩立面色凝重的从储物袋中取出那枚聚灵丹，稍微把玩了一下，就不再犹豫的一口服下。
> 
> 顿时腹中升起一股热流，向四肢百骸扩散开来。韩立不敢怠慢，急忙盘膝坐好，按照长春功的路线，控制着这股热流在体内缓缓流动。
> 
> 三个周天之后，热流随着功法的运转，开始慢慢融入到经脉之中，而长春功的运转速度，也比平日里快了近半。
> 
> 如此又过了一顿饭的功夫，当韩立再次催动功法时，体内的瓶颈竟真的松动了一下。
> 
> “有戏！”韩立心中一喜，更加小心谨慎的控制着热流，向那层看不见的瓶颈冲击而去。

### 场景：获得宝物

原文：
> 韩立神识在玉简上略一流转，心中就已明了，这是一套名为"凝水功"的顶阶功法。
> 
> 他不动声色的将神识退出，把玩了两下玉简后，才淡淡的向对方问道：
> 
> “代价呢？"
> 
> 这么简单的一句问话，却让青年的笑容为之一僵，他原本准备好的诸多说辞，竟一时不知该如何开口。

## 写作技巧

### 主角塑造

1. **资质平庸**：韩立是杂灵根，修炼速度慢，但靠勤奋和机遇弥补
2. **谨慎低调**：不到万不得已不出手，出手必解决问题
3. **精于算计**：每做一件事都会权衡利弊，有后手

### 升级体系

```
炼气（1-13层）→ 筑基（初期/中期/后期/大圆满）→ 金丹 → 元婴 → 化神 → 炼虚 → 大乘 → 渡劫
```

### 爽点设计

| 类型 | 出现场景 |
|------|----------|
| 越级杀敌 | 筑基杀金丹 |
| 宝物收获 | 遗迹寻宝 |
| 秘法突破 | 功法变异 |
| 势力扩张 | 建立宗门 |

### 经典套路

1. **资源收集**：炼丹、炼器、收集材料
2. **秘境探险**：上古遗迹、洞府传承
3. **势力争斗**：宗门大战、种族对决
